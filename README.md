# my phoenix config

### Installation

1. Install Phoenix
   ```sh
   brew cask install phoenix.
   ```
2. In `~/.config` run:
   ```sh
   git clone https://github.com/tborychowski/phoenix-config.git phoenix
   ```
3. Start Phoenix
